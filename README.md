# 🧩 PUZZLE

An interactive, customizable image-puzzling application built entirely in Python. Transform any personal photo into a custom jigsaw puzzle, choose your difficulty, and start solving!

---

## ✨ Features

* **Custom Image Upload:** Choose any picture from your local device to turn into a puzzle.
* **Dynamic Grid Slicing:** You control the difficulty! Cut your picture into any number of rows and columns you like.
* **Free-Form Solving:** Drag, drop, and piece your image back together freely.
* **Lightweight Stack:** Built purely in Python for quick setup and easy execution.

---

## 🛠️ Tech Stack

* **Language:** Python 3.2
* **Core Libraries:** *Tkinter, PIL, os, random, sys*
